🌿 Shvil Hazhav – שביל הזהב

אפליקציית Android חכמה וידידותית שנועדה לסייע לאנשים מבוגרים (60+) ובעלי מוגבלויות בניהול תרופות, חיזוק הזיכרון, תקשורת חברתית וקבלת מידע רפואי – במקום אחד.

🎯 מטרת הפרויקט

הפרויקט נועד לתת מענה לצורך יומיומי קריטי:
מניעת מצבי חירום רפואיים הנגרמים משכחת נטילת תרופות, תוך חיזוק הזיכרון ושיפור איכות החיים של האוכלוסייה המבוגרת.

האפליקציה משלבת:

תזכורות חכמות לנטילת תרופות

חיזוק יכולות קוגניטיביות באמצעות משחק

תקשורת חברתית בין משתמשים

גישה לסוכן AI בנושאים רפואיים

👥 קהל יעד

אנשים מבוגרים (60+)

בעלי מוגבלויות הזקוקים לתמיכה יומיומית

תזכורות אוטומטיות

🧠 משחק זיכרון אונליין

משחק התאמות (Memory Game)

משחק אונליין אחד-על-אחד

ניקוד:

התאמה = +1 נקודה

טעות = −1 נקודה

3 סבבים בכל משחק

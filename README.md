<h1>🌿 Shvil Hazhav – שביל הזהב</h1>

<p>
אפליקציית <b>Android חכמה וידידותית</b> שנועדה לסייע לאנשים מבוגרים (60+) ובעלי מוגבלויות
בניהול תרופות, חיזוק הזיכרון, תקשורת חברתית וקבלת מידע רפואי – במקום אחד.
</p>

<hr>

<h2>🎯 מטרת הפרויקט</h2>
<p>
הפרויקט נועד לתת מענה לצורך יומיומי קריטי:
<b>מניעת מצבי חירום רפואיים הנגרמים משכחת נטילת תרופות</b>,
תוך חיזוק הזיכרון ושיפור איכות החיים של האוכלוסייה המבוגרת.
</p>

<p>האפליקציה משלבת:</p>
<ul>
  <li>תזכורות חכמות לנטילת תרופות</li>
  <li>חיזוק יכולות קוגניטיביות באמצעות משחק</li>
  <li>תקשורת חברתית בין משתמשים</li>
  <li>גישה לסוכן AI בנושאים רפואיים</li>
</ul>

<hr>

<h2>👥 קהל יעד</h2>
<ul>
  <li>אנשים מבוגרים (60+)</li>
  <li>בעלי מוגבלויות הזקוקים לתמיכה יומיומית</li>
  <li>משתמשים הזקוקים לממשק פשוט, ברור ונגיש</li>
</ul>

<hr>

<h2>🧩 מודולים מרכזיים</h2>

<h3>💊 ניהול תרופות</h3>
<ul>
  <li>הזנת תרופות בעת הרישום ולאחר מכן בכל עת</li>
  <li>קביעת מועדי נטילה</li>
  <li>תזכורות אוטומטיות</li>
  <li>מחיקת תרופות שפג תוקפן עם חיווי למשתמש</li>
</ul>

<h3>🧠 משחק זיכרון אונליין</h3>
<ul>
  <li>משחק התאמות (Memory Game)</li>
  <li>משחק אונליין אחד-על-אחד</li>
  <li>
    ניקוד:
    <ul>
      <li>התאמה = +1 נקודה</li>
      <li>טעות = −1 נקודה</li>
    </ul>
  </li>
  <li>3 סבבים בכל משחק</li>
  <li>קביעת מנצח לפי מספר נקודות מצטבר</li>
</ul>

<h3>👥 פורום ותקשורת בין-אישית</h3>
<ul>
  <li>יצירת קשר בין משתמשים</li>
  <li>שיתוף ידע רפואי וחוויות</li>
  <li>חיזוק תחושת השייכות והקהילתיות</li>
</ul>

<h3>🤖 סוכן AI רפואי</h3>
<ul>
  <li>ממשק שאלות-תשובות</li>
  <li>שאילתות בנושאים רפואיים כלליים</li>
  <li>מענה ברור ופשוט המותאם לאוכלוסייה מבוגרת</li>
</ul>

<p><b>⚠️ שים לב:</b> הסוכן אינו מהווה תחליף לייעוץ רפואי מקצועי.</p>

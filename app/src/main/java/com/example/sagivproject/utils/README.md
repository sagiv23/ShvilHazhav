# Utils

> Static utility classes and helper methods used across the application.

---

| Utility                 | Purpose                                                                |
|-------------------------|------------------------------------------------------------------------|
| `ImageUtil`             | Methods for bitmap manipulation, base64 encoding, and image processing |
| `Validator`             | Input validation logic (e.g., email format, password strength)         |
| `CalendarUtil`          | Date and time formatting and calculation helpers                       |
| `SharedPreferencesUtil` | Easy access to local persistent storage for user preferences           |

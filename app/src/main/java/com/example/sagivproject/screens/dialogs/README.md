# Dialogs

> Popup UI components for user interaction and data entry.

---

| Dialog                    | Purpose                                             |
|---------------------------|-----------------------------------------------------|
| `AddUserDialog`           | Form to create a new user account (Admin only)      |
| `EditUserDialog`          | Form to update existing user information            |
| `ConfirmDialog`           | General confirmation prompt for destructive actions |
| `MedicationDialog`        | Entry form for adding or editing medication details |
| `FullImageDialog`         | Full-screen image preview                           |
| `ProfileImageDialog`      | UI for selecting or changing profile photos         |
| `EditForumCategoryDialog` | Management of forum categories                      |

# Screens

> Activities that handle user interaction and orchestrate the application's primary features.

---

| Activity                             | Description                                                                    |
|--------------------------------------|--------------------------------------------------------------------------------|
| `AdminPageActivity`                  | Central dashboard for administrators to access management tools.               |
| `AiActivity`                         | Interactive chat interface with a generative health AI (Gemini).               |
| `ContactActivity`                    | Displays developer info and links, including a secret navigation trigger.      |
| `DetailsAboutUserActivity`           | View and manage personal profile data, including profile image updates.        |
| `EmergencyContactsActivity`          | Manage emergency contacts and trigger manual/automatic SOS alerts.             |
| `ForumActivity`                      | Real-time discussion board within a specific category.                         |
| `ForumCategoriesActivity`            | Browsing and administration of forum discussion topics.                        |
| `GameHomeScreenActivity`             | Matchmaking lobby and personal stats for the memory game.                      |
| `LandingActivity`                    | Onboarding screen for unauthenticated users.                                   |
| `LoginActivity` / `RegisterActivity` | User authentication and account creation flows.                                |
| `MainActivity`                       | Primary navigation hub for standard users.                                     |
| `MathProblemsActivity`               | Mental stimulation tool with randomized arithmetic challenges.                 |
| `MedicationImagesTableActivity`      | Administrative grid for managing memory game card assets.                      |
| `MedicationListActivity`             | Central management for medication prescriptions and daily intake logging.      |
| `MemoryGameActivity`                 | Live 1-on-1 multiplayer memory game session.                                   |
| `MemoryGameLogsTableActivity`        | Real-time administrative log of all game rooms and sessions.                   |
| `SettingsActivity`                   | Application preferences including Dark Mode and Fall Detection toggle.         |
| `SplashActivity`                     | Launch screen handling session validation and initial routing.                 |
| `TipOfTheDayActivity`                | Displays daily advice, either from the database or AI-generated.               |
| `UserStatsActivity`                  | Performance analytics and history logs, with admin cross-user viewing support. |
| `UsersTableActivity`                 | Administrative management of the user database and roles.                      |

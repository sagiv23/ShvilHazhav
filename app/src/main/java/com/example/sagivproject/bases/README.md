# Bases

> Base classes and interfaces that provide common functionality.

---

| Class                 | Type           | Purpose                                                      |
|-----------------------|----------------|--------------------------------------------------------------|
| `BaseAdapter`         | Abstract Class | Base logic for RecyclerView adapters                         |
| `BaseFragment`        | Abstract Class | Common setup for all fragments (e.g., DI, UI initialization) |
| `BaseForumFragment`   | Abstract Class | Specialized base for forum-related fragments                 |
| `BaseDatabaseService` | Abstract Class | Shared base logic for database access                        |

# UI Components

> Reusable UI components, fragments, and specialized display helpers.

---

| Component                | Purpose                                                     |
|--------------------------|-------------------------------------------------------------|
| `AdminMenuFragment`      | Fragment for administrative navigation menu                 |
| `LoggedInMenuFragment`   | Fragment for standard user navigation menu (when logged in) |
| `LoggedOutMenuFragment`  | Fragment for guest navigation menu                          |
| `SimpleXYGraphView`      | Custom view for rendering simple performance graphs         |
| `CustomTypefaceSpan`     | Custom font styling for `Spannable` strings                 |
| `MenuNavigationListener` | Interface for handling menu navigation events               |

# Adapters

> RecyclerView adapters that bind model data to list item views.

---

| Adapter                        | Model           | Purpose                                         |
|--------------------------------|-----------------|-------------------------------------------------|
| `ForumAdapter`                 | `ForumMessage`  | Displays forum messages in a list               |
| `MemoryGameAdapter`            | `Card`          | Manages the grid of cards for the memory game   |
| `UsersTableAdapter`            | `User`          | Displays a table of users for admin management  |
| `LeaderboardAdapter`           | `User`          | Displays top users sorted by their scores       |
| `ForumCategoryAdapter`         | `ForumCategory` | Displays different categories in the forum      |
| `MemoryGameLogAdapter`         | `GameRoom`      | Displays history and logs of memory games       |
| `MedicationListAdapter`        | `Medication`    | Displays a list of medications for the user     |
| `MedicationImagesTableAdapter` | `ImageData`     | Displays a gallery of medication-related images |

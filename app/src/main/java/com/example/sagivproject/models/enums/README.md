# Enums

> Enumerated types that define a fixed set of constants.

---

| Enum | Description |
|------|-------------|
| `UserRole` | Defines roles for users (e.g., ADMIN, USER) |
| `Operation` | Represents different operations in the system (e.g., ADD, EDIT, DELETE) |
| `MedicationType` | Categorizes medications (e.g., PILL, LIQUID) |

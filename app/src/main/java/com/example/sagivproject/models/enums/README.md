# Enums

> Enumerated types that define a fixed set of constants used throughout the application.

---

| Enum               | Description                                                                         |
|--------------------|-------------------------------------------------------------------------------------|
| `MedicationStatus` | Tracks the intake status of a medication dose (TAKEN, NOT_TAKEN, SNOOZED).          |
| `MedicationType`   | Categorizes the physical form of medications (PILL, SYRUP, CREAM, etc.).            |
| `Operation`        | Defines mathematical operations for the math challenge (ADD, MULTIPLY, SQRT, etc.). |
| `UserRole`         | Defines user account privileges within the system (ADMIN, REGULAR).                 |
